# py-webgrabber
Python tool to grab screenshots of webpages from a list

```
usage: webgrabber.py [-h] --list LIST --driver DRIVER --browser BROWSER [--out OUT] [--verbose VERBOSE]

WebGrabber

optional arguments:
  -h, --help            show this help message and exit
  --list LIST, -l LIST  file list of domains to run WebGrabber against
  --driver DRIVER, -d DRIVER
                        /path/to/chromedriver
  --browser BROWSER, -b BROWSER
                        /path/to/chrome
  --out OUT, -o OUT     output directory (default output/, will be created if it does not exist)
  --verbose VERBOSE, -v VERBOSE
                        enable verbose output
```
